# ReciFlowLite

A lightweight iOS recipe recording app designed for fast, on-site use.

ReciFlowLite focuses on rapid ingredient input and recall, optimized for real-world cooking workflows rather than decorative recipe browsing.

## Features
- High-speed ingredient input optimized for daily use
- Vertical / horizontal navigation for fast editing
- Unified input flow without modal interruptions
- Local-first data storage with export support
