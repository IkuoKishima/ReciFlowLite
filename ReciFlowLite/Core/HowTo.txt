プロパティ（property）

「その物が持っている状態・値」

例：recipes / isLoading / pendingUndo / engineStores

「装備や設定」に近い感覚で合ってる。ただしGUIじゃなくて“メモリ上の変数”として存在する。

メソッド（method）

「その物ができる行動（処理・手順）」

例：loadRecipes() / requestDelete(...) / updateRecipeMeta(...)

「削除・追加もメソッド？」→ はい。
“削除という挙動”をコードにしたものがメソッド。

イニシャライザ（initializer / init）

「生まれた直後に何をするか」

init() { loadRecipes() }

型・クラス（class）

final class RecipeStore: ObservableObject { ... }

「RecipeStore という箱（役割）を作る宣言」

属性（attribute）

@Published, @MainActor, @discardableResult

「このプロパティ/メソッドに追加ルールを付けるラベル」


@Published は「この値が変わったら、画面（View）更新してね」というSwiftUI向けラベル。

private は「外から触らせない（RecipeStoreの中だけで使う）」という境界線。
