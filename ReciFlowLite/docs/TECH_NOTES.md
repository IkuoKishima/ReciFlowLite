# Tech Notes: release-1.1.1

## Navigation / Dock
- 目的：時間縛り入力テストにおいて、指と視線の移動距離が直感的になるようボタン配置を最適化
- 方針：新機能は入れずに位置の変更と調整だけで最適化を図る
安定化：
- ボタン配置テスト中の強い負荷で発生した再現不可能な揺らぎに対し、ガードを追加
- 現状のボタン表現をリキッドデザインで統一
- ナビゲーションドックとライトドックは構造上同一だが、Appleの制約により再現不可能な演出は今回除外
影響範囲：RightRail（UIKit / コントロール系）


# Tech Notes: release-1.1.0

## Navigation / Dock
- 目的：入力の流れを止めずに、素早く移動できる導線を確保する
- 方針：Dockは「使わなくても良い」補助UI。主要導線を邪魔しない
- 安定化：フォーカス更新のループ防止、遷移前のフォーカス解除、キーボードdismissの統一
- 影響範囲：IngredientEngine（縦遷移/横遷移/レール）
